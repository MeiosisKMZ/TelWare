# TelWare
PoC of a program to reboot indefinetly a vulnerable printer using exposed unprotected Telnet access

> [!WARNING]
> Please, this a Proof of Concept, do not test against stuff you don't own or are explicitly allowed to test. 

May not work on your printer but it worked on my old HP JetDirect.
Be careful, it can be considered as a malware as it will put your printer in a reboot loop, causing a DoS.
